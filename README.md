# YouTube Clone ( MERN stack project)
```
A simple YouTube clone application built using the MERN stack (MongoDB, Express, React, Node.js). This project replicates the essential features of YouTube, including video uploads, search functionality, user authentication, and a video player.

Features
User Authentication: Sign up, login, and session management using JWT.
Video Uploads: Users can upload their videos to the platform.
Video Playback: Stream videos using a built-in player.
Search Functionality: Users can search for videos by title or keywords.
Responsive Design: Works on both desktop and mobile devices.
Like and Comment: Interact with videos by liking or commenting on them.
Project Setup
Prerequisites
Node.js (v14.x or higher)
MongoDB (Running locally or using MongoDB Atlas)
NPM (Node Package Manager)

# Clone the repository:
git clone https://github.com/yourusername/youtube-clone.git
cd youtube-clone

## SETUP

Backend Setup:
Navigate to the backend directory:
cd backend
Install the dependencies:
npm install
Set up environment variables in a .env file:
MONGODB_URI=your-mongodb-uri
JWT_SECRET=your-jwt-secret
PORT=5000
Start the backend server:
npm start

Frontend Setup:
Navigate to the frontend directory:
cd frontend
Install the dependencies:
npm install
Start the frontend development server:
npm start
Running the Application:
Backend should be running on http://localhost:5000.
Frontend should be accessible at http://localhost:3000.
Usage:
Sign up to create a new account.
Login to access features like uploading videos, liking videos, and commenting.
Upload videos and share them with other users.
Search for videos by title, and interact with them by liking or commenting.
Technologies Used:
Frontend: React, Redux, React Router, Axios
Backend: Node.js, Express, JWT ,cors, bcrypt
Database: MongoDB(Backend) , React(Frontend)
